Books

These are books related to Plan 9, its ideas, its design and its style.
  • The Elements of Style William Strunk, Jr. E. B. White (Online version of the first edition)
  • The Mythical Man-Month: Essays on Software Engineering by Frederick P. Brooks.
  • Lions' Commentary on UNIX by John Lions. (Ps and Pdf versions of the book; prepared source code for v6 Unix: Ps and Postscript and HTML versions)

PROGRAMMING

  • The Unix Programming Environment by Brian W. Kernighan and Rob Pike. ISBN 0-13-937681-X (paperback), 0-13-937699-2 (hardback).
  • The Practice of Programming by Brian W. Kernighan and Rob Pike. ISBN 0-201-61586-X.
  • The C Programming Language by Brian W. Kernighan and Dennis M. Ritchie. ISBN 0-13-110362-8 (second edition, paperback)
  • The AWK Programming Language by Alfred V. Aho, Brian W. Kernighan and Peter J. Weinberger. ISBN 0-201-07981-X.
  • Communicating Sequential Processes (CSP) by Tony Hoare - CSP is the concurrency model used in Inferno and thread(2) - For more information see http://www.usingcsp.com
